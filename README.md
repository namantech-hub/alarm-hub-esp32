# Intrusion alarm system

Intrusion alarm system

## Requirements

### Perform functions -
1. arm
2. home arm
3. disarm
4. Add / Remove LoRa sensor
5. Add / Remove RFID card
6. Handle Power on/off/restart from PIC16F
7. Measure battery voltage & mains voltage
8. Set arm/disarm/panic sounds
9. Play music
10. Handle LEDs
11. Process signal received from -
    a. Ble
    b. mqtt
    c. Lora
    d. HTTP / WS API
12. Perform LoRa repeater function
